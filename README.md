# MUSIIMENTA CISSYLYN — Portfolio (scaffold)

This repository is a premium portfolio scaffold built with React + TypeScript + Tailwind CSS, Framer Motion and Three.js ready for enhancements.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

- [src/App.tsx](src/App.tsx)

- [src/components/Projects.tsx](src/components/Projects.tsx)
